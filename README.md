## 🔹 Descripción

**SISSEP** (Sistema Institucional de Seguimiento de Servicio y Residencias Profesionales) es una plataforma web institucional que centraliza, gestiona y da seguimiento a los documentos requeridos para el **servicio social** y las **residencias profesionales**.  

Permite a estudiantes, tutores y administradores mantener un flujo de trabajo organizado y reducir errores manuales.

---

## 🔹 Características principales

- Gestión de usuarios (estudiantes, tutores, administradores)  
- Registro y seguimiento de documentos requeridos  
- Interfaz web moderna con tablas y formularios  
- API REST con Node.js y Express  
- Base de datos PostgreSQL segura y eficiente  
- Frontend con React + TypeScript
